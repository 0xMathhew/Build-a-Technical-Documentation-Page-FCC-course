C++ Introduction

 C++ is a powerful and versatile programming language that has been widely used in software development for several decades. It was developed as an extension of the C programming language with added features for object-oriented programming (OOP). C++ combines the efficiency and low-level control of C with high-level features that enable developers to create complex and efficient software systems.

 C++ is used in a wide range of applications, including systems programming, game development, embedded systems, scientific computing, and more.

Loops

What is a loop?

Loop is a defined set of instructions that will execute until a condition is true.

        #include <iostream>
          int main {
            for (int i{}; i < 10; i++) // for loop example
            std::cout << i << '/n';
            return 0;
          }
Conditional Statements

What are conditional statements?

If a condition evaluates to true/false then appropriate set of instructions is executed. 

        #include <iostream>
          int main {
            if (1 = 2){
              std::cout << "If true this will be executed" << '/n';           
            } else {
              std::cout << "If false this will be executed" << '/n';
            }
            return 0;
          }
Arrays

What is array?

Array is a bunch of variables stacked together in one object. So you can have 10 int variables in one array.

        #include <iostream>
          int main {
            int x[10];
            int x[2] = 11;
            std::cout << x[2] << '/n';
            return 0;
          }
Pointers

What's a pointer?

Pointer is a memory address of something. It can be a variable. It is faster than accesing it the normal.

        #include <iostream>
          int main {
            int x{4};
            int *pointer_variable = &x; // pointer that get assigned address of x
            std::cout << &x << '/n';
            return 0;
          }
